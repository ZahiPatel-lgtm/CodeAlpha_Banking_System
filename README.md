## Banking System

This project is a simple Banking System developed in C++ using Object-Oriented Programming (OOP) concepts.  
The program allows users to create customer accounts, perform deposits, withdrawals, fund transfers and check account balances through a menu-driven interface.

### Features
- Create Customer and Account
- Deposit Money
- Withdraw Money
- Fund Transfer
- Display Account Balance
- Simple menu-driven interface
- Beginner-friendly implementation

### Concepts Used
- Classes and Objects
- Functions
- Conditional Statements
- Loops
- User Input and Output
- Basic Banking Operations
